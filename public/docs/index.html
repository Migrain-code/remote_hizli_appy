Introduction

Base URL: http://127.0.0.1:8001

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Authentication

POST api/auth/login

Status Codes

  • phone
  • password
  • 401 Unauthorized Hatası

Login apisi

Example request:
curl --request POST \
    "http://127.0.0.1:8001/api/auth/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"phone\": \"eius\",
    \"password\": \"i\\/#};-!;e*BkKm#Cxa\"
}"

const url = new URL(
    "http://127.0.0.1:8001/api/auth/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "phone": "eius",
    "password": "i\/#};-!;e*BkKm#Cxa"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Headers

Content-Type      

Example: application/json

Example: application/json

Body Parameters

name   string   

value değeri 255 karakter değerinden küçük olmalıdır. Example: gcwblerwedaayjtklxywytyls

number   number   

value değeri 16 değerinden büyük olmalıdır. Example: 65

month   number   

value 1 - 12 arasında olmalıdır. Example: 1

year   number   

Example: 2874672.07575

cvc   string   

Must match the regex /^\d{3,4}$/. Example: 9597

name_on_the_card   string   

value değeri 255 karakter değerinden küçük olmalıdır. Example: qzmk

is_default   number   

Example: 1

Must be one of:
  • 0
  • 1
